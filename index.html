Dictionary Definition Analysis Tool

About this tool

This tool was inspired by Urban Dictionary Definitions of my name. I found that many of them had similar themes so I wanted to see if I could build a tool to discover these themes.

Currently the tool does a few things:

  • Determines relative word and sentence importances.
    1. Works out the most commonly used words across all definitions,
    2. then finds the whole sentences across all definitions that use this works the most.
  • Displays all the words above 9 characters in length.
  • Displays all the words below 4 characters in length.

To use, just search for the word you'd like to analyse. You'll be taken to a results page on AWS.
